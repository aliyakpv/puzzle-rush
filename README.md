## Puzzle rush
This personal project is snake game 

## To start
1. Clone the Repository
2. Navigate to the Project Directory
3. Configure and generate build files
4. Build the project
5. Run the executable

```bash
cmake -S . -B build
cd build
make
./puzzle_rush
```
